# Memora 应用图标设置指南

## 图标设计建议

基于"note"主题，建议采用以下设计元素：

### 🎨 设计概念
- **主元素**：笔记本或纸张图标
- **配色方案**：
  - 主色：科技橙 (#FF9500)
  - 渐变：从浅橙到深橙的渐变效果
  - 可选：添加发光效果增强科技感
- **风格**：现代简约、圆角设计，符合macOS/iOS设计规范

### 📐 图标尺寸要求

macOS 需要以下尺寸：
- 16x16 (1x, 2x)
- 32x32 (1x, 2x)
- 128x128 (1x, 2x)
- 256x256 (1x, 2x)
- 512x512 (1x, 2x)

iOS 需要添加：
- 20x20, 29x29, 40x40, 60x60, 76x76, 83.5x83.5, 1024x1024

## 方法一：使用在线工具生成（推荐）

### 1. 使用 AppIcon.co
1. 访问 https://www.appicon.co
2. 上传一张 1024x1024 的设计图
3. 选择 iOS 和 macOS 平台
4. 下载生成的图标包
5. 将文件拖入 Xcode 的 AppIcon.appiconset

### 2. 使用 SF Symbols（快速方案）
如果想快速使用系统图标：

```swift
// 临时方案：在 Info.plist 中设置
// 添加 CFBundleIconName 键，值为 "AppIcon"
// 或使用 SF Symbol 作为占位图标
```

## 方法二：自己设计图标

### 推荐设计工具
- **Figma**（免费，在线）
- **Sketch**（macOS专业工具）
- **Canva**（简单易用）
- **Adobe Illustrator**（专业设计）

### 设计步骤

1. **创建 1024x1024 画布**
   - 确保足够的分辨率
   - 使用矢量图形

2. **设计元素示例**
   ```
   ┌─────────────────┐
   │  ╔═══════════╗  │
   │  ║  Memora   ║  │  ← 简洁的笔记本样式
   │  ║  ─────    ║  │  ← 横线代表文字
   │  ║  ─────    ║  │
   │  ╚═══════════╝  │
   └─────────────────┘
   ```

3. **应用渐变和阴影**
   - 使用橙色渐变（#FFB84D → #FF9500）
   - 添加微妙的内阴影增加立体感
   - 考虑添加发光效果（外发光）

4. **导出图标**
   - 导出 1024x1024 PNG
   - 使用图标生成工具生成所有尺寸

## 方法三：在 Xcode 中直接设置

1. 打开 Xcode
2. 在项目导航器中找到 `Assets.xcassets`
3. 点击 `AppIcon`
4. 将不同尺寸的图标拖入对应的方框中

## 快速占位方案（开发阶段）

如果暂时不想设计图标，可以：

1. 使用 SF Symbols 的 "note.text" 图标
2. 在 Sketch/Figma 中快速创建一个橙色背景 + 白色笔记本图标
3. 使用渐变填充（橙色系）+ 简单几何形状

## 推荐的简单设计

```
背景：线性渐变
  顶部：#FFB84D
  底部：#FF8800

前景：白色笔记本图标
  - 圆角矩形（代表笔记本）
  - 3-4条横线（代表文字）
  - 左上角小圆点（装饰元素）

效果：科技感、简洁、现代、温暖
```

## 注意事项

- ✅ 使用高对比度，确保在暗黑模式下也清晰
- ✅ 避免过多细节，图标要在小尺寸下也清晰可辨
- ✅ 保持圆角，符合 Apple 设计规范
- ✅ 测试在不同背景色下的显示效果
- ❌ 避免使用文字（除非非常大且清晰）
- ❌ 避免使用过于复杂的渐变

## 完成后

设置完图标后，在 Xcode 中：
1. Clean Build Folder (⇧⌘K)
2. 重新编译运行
3. 在 Dock 和启动台查看效果
